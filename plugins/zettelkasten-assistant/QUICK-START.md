# Zettelkasten Assistant - Quick Start

## Command Reference

### Daily Commands

**Capture a thought**:
```
/capture I think RAG could benefit from chain-of-thought
```

**Process inbox** (5-10 min):
```
/process-inbox
```

### Weekly Commands

**Process fleeting notes** (30 min):
```
/process-fleeting
```

**Link new notes**:
```
/link-notes permanent/ai-ml/transformer.md
```

### Organizational Commands

**Create Map of Content**:
```
/create-moc reinforcement learning
```

## Agent Invocation

Agents run autonomously - just ask naturally:

**Process all notes**:
```
Process all my notes
Help me clean up my captures
```

**Find connections**:
```
Find connections between my notes
Discover relationships in my vault
```

**Update MOCs**:
```
Update my MOCs
Maintain my maps of content
```

## Tips

1. **Tab completion**: Type `/zette` to filter commands
2. **Arguments**: Most commands work with or without arguments
3. **Natural language**: For complex tasks, just ask - agents will trigger automatically
4. **Daily rhythm**: Capture freely, process daily, review weekly
